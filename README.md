## Setup

1/ Install Docker

2/ Install Elixir and Erlang

3/ Fetch dependecies

```bash
mix deps.get
```

4/ Boostrap the environment

```bash
bin/envsetup.sh
```

## Start the Phoenix server

```bash
mix phx.server
```

The Web app will run on `localhost:4000`
